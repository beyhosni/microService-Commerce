# Angular Shop (Frontend)
Frontend minimal (Angular 17) pour l'e-commerce.
## Dev
- Node.js >= 20
- `npm install`
- `npm start` (http://localhost:4200, proxy `/api` vers Gateway)
## Docker
- `docker build -t angular-shop:latest .`
